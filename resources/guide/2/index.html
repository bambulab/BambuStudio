User Agreement
The main interface of the system is divided into 8 areas: 1. Enterprise brand logo area: The enterprise brand logo is displayed. The administrator can change the logo of the login interface and internal interface in the [system setting] à [system core setting] à [interface setting] function. 2. To do / reminder shortcut button area: There will be numbers next to these buttons to indicate your current number of reminders / to-do items. Click to quickly enter the reminder / to-do interface to view relevant contents. 3. Help button: Click the help button to see the number of people online, the current version and the instructions in the help 4. Navigation menu area: At the menu entry of all functions of the system, click the corresponding function menu to open the corresponding function interface. 5. The function quick switch label area has been opened: If you open multiple functions at the same time, you can quickly switch different functions in this area to truly realize multi-function and multi task at the same time. Tip: the maximum number of function labels that can be opened is determined by the system setting. The specific number of function labels that can be opened can be set by the administrator in [system setting] à [system core setting] à [interface setting] à [maximum number of open labels] 6. Main work interface area: The main working interface, display / operation of most functions and other elements will be displayed in this area. 7. Personal login information display area: Display the current login user name (account). Click to exit the account and modify your own information password. 8.
